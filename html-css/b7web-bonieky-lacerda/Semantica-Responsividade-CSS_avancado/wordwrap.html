WordWrap

Propriedade WordWrap é utilizada quando o texto estoura a div. Usando a propriedade: word-wrap:break-word o texto que estoura a div é mantido dentro da div de forma visual.

Na sendo necessário o uso da propriedade: overflow: hidden; para esconder o estouro da div. A solução correta seria usar, word-wrap: break-word;
